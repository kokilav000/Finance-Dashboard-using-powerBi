# Finance-Dashboard-using-powerBi

##  Project Overview
This project is an **interactive Finance Dashboard** built using **Power BI Desktop** as part of an **assignment for a data analytics training program**.  
The dashboard analyzes sales, costs, profit, and units sold across multiple dimensions such as **year, country, segment, product, and month**.
The objective of this assignment was to apply Power BI concepts learned during training to a real-world financial dataset and derive meaningful business insights.

##  Project Objectives
- Apply Power BI skills learned during training  
- Analyze overall financial performance  
- Track sales, COGS, profit, and units sold  
- Create interactive dashboards for business reporting  

##  Training Context
- **Project Type:** Training Assignment  
- **Purpose:** Hands-on practice of Power BI concepts  
- **Skills Applied:** Data cleaning, data modeling, DAX, and visualization  

##  Dataset & Data Model
- Financial sales dataset provided as part of the training assignment  
- Multiple related tables including Orders, Financials, People, and Returns  
- Data cleaned and transformed using **Power Query**

## Tools & Technologies
- Power BI Desktop  
- Power Query  
- DAX  
- Excel / CSV  

##  Key Metrics (KPIs)
- Sum of Sales  
- Sum of COGS  
- Sum of Units Sold  
- Sum of Profit  
- Profit Margin  

##  How to Use
1. Download the `financedashboard.pbix` file from this repository  
2. Open it using **Power BI Desktop**  
3. Use slicers and visuals to explore financial insights across different dimensions  

##  Conclusion
This Power BI Finance Dashboard successfully demonstrates the practical application of data analytics concepts learned during training. By transforming raw financial data into interactive and insightful visualizations, the dashboard enables effective analysis of sales performance, cost structure, and profitability. This project highlights hands-on experience with Power BI, DAX, and data modeling, and supports data-driven decision-making.
